Pasos para actulizar el repo:

1.- Lo clonamos con "git clone https://github.com/m4jbz/dotfiles".

2.- Hacemos los cambios que tengamos que hacer (borar, editar, etc.).

3.- Usamos "git add ." para añadir todos los documentos en la actualizacions o "git add 'file'" para solo añadir un archivo o carpeta.

4.- "git commit -m 'update'" para hacer los cambios.

5.- Como siguiente comando sigue "git push origin" para hacer un push en la rama que especificamos.
